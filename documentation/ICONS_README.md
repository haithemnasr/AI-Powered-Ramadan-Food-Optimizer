# PWA Icons Instructions 🎨

To make your app installable, you need to create icon files.

## Quick & Easy Way:

### Option 1: Use Online Tool (Easiest!)
1. Go to: https://www.pwabuilder.com/imageGenerator
2. Upload any logo/image (at least 512x512px)
3. Click "Generate"
4. Download the zip file
5. Extract and use `icon-192.png` and `icon-512.png`

### Option 2: Create Simple Icons
1. Open any image editor (Canva, Photoshop, even Paint!)
2. Create a 512x512px image with:
   - Background: Dark blue (#1a1f2e)
   - Text: "B9itlek" in gold (#ffd700)
   - Add moon emoji 🌙 or plate emoji 🍽️
3. Save as `icon-512.png`
4. Resize to 192x192px and save as `icon-192.png`

### Option 3: Use Emoji as Icon
1. Go to: https://favicon.io/emoji-favicons/
2. Choose 🌙 or 🍽️ emoji
3. Download and rename to `icon-192.png` and `icon-512.png`

## File Requirements:
- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels
- Both should be PNG format
- Place them in the same folder as index.html
